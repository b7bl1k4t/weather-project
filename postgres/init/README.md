# Init Scripts

Laravel теперь управляет схемой базы данных через миграции (`php artisan migrate`).  
Каталог оставлен пустым, потому что `postgres` запускает все `.sql`/`.sh` в этой директории при первом старте контейнера. Если понадобится дополнительная инициализация на стороне БД, поместите отдельный SQL-файл сюда.
